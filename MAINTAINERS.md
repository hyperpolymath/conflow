## Responsibilities

Maintainers are expected to:

1. **Review Contributions**
   - Respond to MRs within 1 week
   - Provide constructive feedback
   - Merge approved changes

2. **Triage Issues**
   - Label and prioritize issues
   - Close stale/duplicate issues
   - Guide contributors

3. **Security Response**
   - Monitor security reports
   - Coordinate vulnerability fixes
   - Manage disclosure timeline

4. **Community Health**
   - Enforce Code of Conduct
   - Welcome new contributors
   - Foster inclusive environment

5. **Release Management**
   - Prepare release notes
   - Tag releases
   - Update documentation
