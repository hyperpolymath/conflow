== RSR Stack

This project follows RSR conventions:

* ✅ ReScript for frontend/logic
* ✅ Deno for JS runtime
* ✅ WASM for performance-critical code
* ✅ Rust/OCaml/Haskell for systems/proofs
* ✅ Guile/Scheme for configuration
* ❌ No TypeScript
* ❌ No Go
* ❌ No npm
